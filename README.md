# VPN_VLESS

Системные требования:  
* 1 gb озу
* 1 core
* 10 gb ssd/hdd
* 100 mbit/sec - скорость интернета

Установка 3x-ui:
```
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
```
* y
* username
* password
* port

Заходим в панель `10.10.201.1:55555` -> `inbounds`
